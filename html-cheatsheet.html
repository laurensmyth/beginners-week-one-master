Headings and Paragraphs

We make headings using heading tags, from h1 down to h6, decreasing in size. Generally h1 is for page titles and h4-h6 are rarely required. Text not in a heading is contained in a paragraph tag.

To start a new paragraph, start a new paragraph tag, this automatically separates your text.

We can also write text in italics

Blank Lines, Void Tags and Comments

We can insert extra blank lines using the br tag, known as the void tag. Note this tag doessn't require a closing tag, only the opening one. Remember to close your paragraph before the break tag and to start a new one after


We can also insert comments. These will show up in the code but not the browser.

List Types

There are two types of lists, ordered which numbers your points and unordered which displays as bullet points. We creat these using the ol and ul tags.
